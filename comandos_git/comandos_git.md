# Comandos do git
## Clonando o repositório
```cmd
git clone '<link repositório>'
```
## Adicionando os arquivos
```cmd
git add '<nome arquivo>' ou '.' para selecionar tudo
```
## Dando o commit
```cmd
git commit -m "<mensagem de alteração>"
```
## Enviando para o repositório
```cmd
git push origin main
```
